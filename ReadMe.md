# GRAPHY

Graphy is a basic Graphing Calculator developed on Turbo C++ as a 10th Grade side project


# INCLUDED

  - The C++ code
  - Screenshots of the UI 
  
### Installation

Requires Turbo C++
Install the dependencies :

```sh
#include<iostream.h>
#include<conio.h>
#include<stdio.h>
#include<string.h>
#include<math.h>
#include<graphics.h>
```
